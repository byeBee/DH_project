# DynamicHotels

### 프로젝트 개요
#### 프로젝트 명 : Dynamic Hotels 
- 인원: 4명
- 제작 기간 : 2022-06-03 ~ 2022-06-15 (약 2주)
- 주요 기능  
	+ 회원,비회원 예약기능 
	+ 채팅상담기능

- Dynamic Busan 이라는 슬로건처럼 부산내에 있는 호텔 예약을 위한 간단한 프로그램 구현
